# 咖啡粉粒徑分析工具 (Coffee Grind Particle Size Analyzer)

本專案是一個基於 Python 與 OpenCV 的咖啡粉粒徑分析軟體。透過數位影像處理技術，能自動偵測照片中的每一顆咖啡粉顆粒，計算其等效圓直徑（Equivalent Circular Diameter）與圓形度，並輸出詳細的粒徑分佈圖表與統計數據。

本工具非常適合用於評估磨豆機的研磨均勻度、分析不同刻度下的粒徑分佈（包含數量分佈與體積分佈，如 D10/D50/D90、Dv10/Dv50/Dv90），協助咖啡愛好者與專業人士優化萃取參數。

---

## 核心功能

1. **自動比例尺與透視校正（推薦模式）**：
   * 支援自動偵測列印於白紙上的正方形校正框（預設 $15\text{cm} \times 15\text{cm}$）。
   * 自動進行透視變換（Perspective Transform）將歪斜的拍照角度攤平成正視圖，修正鏡頭變形。
   * 根據校正框的實際物理邊長，自動計算出精確的比例尺（像素/mm），免去手動量測的麻煩。
2. **多種互動式校正工具**：
   * **手動點選角點**：若照片背景複雜導致自動偵測校正框失敗，可透過 GUI 視窗依序點選四個角進行透視校正。
   * **點選比例尺測量**：若無校正框，可在影像中已知長度的物體上點選兩點，手動輸入實際長度來換算比例尺。
3. **黏連顆粒分離（Watershed 分水嶺演算法）**：
   * 內建自適應局部最大值（Local Maxima）演算法尋找顆粒核心種子，利用分水嶺演算法有效分離互相黏在一起、重疊的粉末顆粒。
4. **過濾與篩選機制**：
   * 自動過濾貼在相片邊界上的不完整顆粒（裁切顆粒）。
   * 支援設定粒徑篩選範圍（微米 $\mu\text{m}$），剔除過小的噪聲或過大的未分離團塊。
   * 可設定圓形度門檻值（Circularity），剔除嚴重變形或低圓形度的多顆粒結塊。
5. **完整的統計與圖表輸出**：
   * **數量粒徑分佈（Number-based Distribution）**：計算 D10、D50（中位數）、D90。
   * **體積粒徑分佈（Volume-based Distribution）**：將顆粒模擬為球體，計算體積權重的 Dv10、Dv50、Dv90。
   * **統計圖表**：自動繪製數量直方圖與體積直方圖（支援中文字型顯示）。
   * **細部數據**：導出每顆顆粒的詳細幾何資訊（面積、直徑、圓形度、中心座標）為 CSV 檔案。
6. **模擬驗證工具（`generate_test_image.py`）**：
   * 可依據指定的 D10/D50/D90 常態分佈參數，自動生成合成顆粒影像（模擬真實咖啡粉）。
   * 自動執行主分析程式並比對「偵測值」與「真實值（Ground Truth）」，產生誤差驗證報告。

## 拍照建議（關鍵步驟）

影像的拍照品質會直接決定分析的準確度，請參考以下建議進行拍攝：

1. **列印正方形校正框**：
   * 本專案已附帶 A4 標準底板 [coffee_calibration_square_15cm.pdf](file:///c:/python_prj/particleAnalyzer/coffee_calibration_square_15cm.pdf)，請直接以 **100% 實際大小（不進行任何縮放）** 列印於 A4 白紙上。列印後的黑色正方形外框邊長即為精確的 $15\text{cm} \times 15\text{cm}$。
2. **均勻撒粉**：
   * 將咖啡粉「均勻攤平、儘量避免重疊結塊」地散布在正方形框**內部**。
   * 由於白紙背景反差足夠，請勿使用過於雜亂的背景。
3. **垂直拍攝**：
   * 相機或手機請垂直由上往下俯拍，確保正方形的四個角都完整入鏡，且光線均勻，避免出現強烈的局部陰影或過曝。
   * 確保校正框邊線沒有被粉末大面積遮擋。

## 使用方式

### 1. 【推薦】使用正方形校正框分析

如果您的照片中有標準的正方形校正框，程式會自動偵測、修正視角並計算比例尺：

```bash
python coffee_grind_analysis.py --image 您的照片路徑.jpg --square-mm 150 --outdir results
```
* `--square-mm`: 正方形校正框的實際物理邊長（單位：mm），預設為 `150.0` (15cm)。
* `--outdir`: 輸出結果存放的資料夾，預設為 `results`。

#### 若自動偵測校正框失敗：
如果背景較複雜導致程式找不到正方形，可加入 `--pick-corners` 參數，在彈出的視窗中依序點選四個角：

```bash
python coffee_grind_analysis.py --image 您的照片路徑.jpg --pick-corners --square-mm 150
```
> *註：互動式視窗需要於本機環境執行（支援顯示視窗的作業系統）。*

---

### 2. 【備用】使用手動比例尺分析（無校正框）

若您沒有印製校正框，但照片中放了尺或其他已知尺寸的物體：

#### A. 測量比例尺（Calibrate 模式）：
執行以下指令，在跳出的視窗中點選已知長度物體的兩端（例如尺上的 1 公分刻度），並在終端機輸入其實際長度（`10` mm），程式將會計算出比例尺：

```bash
python coffee_grind_analysis.py --image 您的照片路徑.jpg --calibrate
```
* 執行後會於終端機輸出類似：`>>> 比例尺 PIXELS_PER_MM = 25.4321`

#### B. 帶入比例尺進行分析：
使用剛剛得到的比例尺（例如 `25.43` px/mm）執行分析：

```bash
python coffee_grind_analysis.py --image 您的照片路徑.jpg --scale 25.43 --outdir results
```

---

### 3. 進階篩選參數

您可以根據顆粒狀況微調以下參數以獲得更精準的結果：

* `--min-diameter-um`: 篩選掉小於此直徑（微米）的粉末（預設 `20`），可用於過濾極細微的背景噪聲。
* `--max-diameter-um`: 篩選掉大於此直徑（微米）的粉末（預設 `2000`），可用於過濾邊緣未分離好的超大團塊。
* `--min-circularity`: 篩選掉圓形度小於此值的顆粒（預設 `0.5`），有效排除多顆黏連卻未被分水嶺拆開的長條狀結塊。
* `--invert`: 若您的咖啡粉顏色「比背景亮」（例如黑紙灑白粉），請加上此參數。

## 輸出結果說明

分析完成後，輸出資料夾（例如 `results/`）下會生成以下檔案：

| 檔案名稱 | 說明 |
| :--- | :--- |
| `annotated.jpg` | **標註影像**：在原圖上用綠色框標記出所有被偵測到的咖啡粉顆粒。 |
| `binary_mask.jpg` | **二值化遮罩**：黑白遮罩圖，可用於檢查二值化是否精準、是否有雜訊。 |
| `square_detection.jpg` | **校正框定位圖**（僅校正框模式）：顯示自動或手動定位的紅線正方形。 |
| `warped.jpg` | **攤平後的影像**（僅校正框模式）：透視變換並裁剪後的正方形內部影像。 |
| `histogram.png` | **數量粒徑分佈直方圖**：以顆粒數量為基準，標示 D10、D50、D90。 |
| `volume_histogram.png` | **體積粒徑分佈直方圖**：以模擬體積為基準，標示 Dv10、Dv50、Dv90。 |
| `particles.csv` | **詳細數據表**：包含每顆顆粒的 ID、面積、直徑、圓形度及中心座標。 |
| `summary.txt` | **統計摘要報告**：包含總顆粒數、平均值、標準差及所有 D-values 指標。 |

## 模擬驗證工具 (`generate_test_image.py`)

為了確保分析程式的分割與計算精度，本專案提供了一個模擬測試指令：

```bash
python generate_test_image.py --num-particles 400 --scale 50.0 --d50 500.0 --d10 350.0 --d90 650.0
```

### 驗證流程：
1. 本指令會隨機產生 400 顆直徑呈常態分佈（Dn50 = $500\mu\text{m}$）的非重疊圓形顆粒。
2. 將其繪製在一個 $2000 \times 2000$ 像素的白色畫布上，保存為 `synthetic_particles.jpg`。
3. 自動調用 `coffee_grind_analysis.py` 對該影像進行分析。
4. 比對「生成時的真實物理尺寸」與「分析程式偵測出的尺寸」，並在控制台印出**詳細比對報表**與誤差百分比（Error %），同時在輸出目錄生成 `validation_report.md`。
